# League Background Changer

Консольная утилита для смены фона профиля в League of Legends на скин из полного актуального каталога игры.

Программа использует локальный League Client API (LCU): клиент League должен быть запущен и пользователь должен войти в аккаунт.

## Быстрый запуск — Node.js не нужен

1. Скачайте папку проекта или только файл `dist/league-bg.exe`.
2. Запустите League of Legends и дождитесь полной загрузки клиента.
3. Откройте CMD в папке проекта и выполните:

   ```bat
   dist\league-bg.exe
   ```

   Также можно дважды нажать `run.bat`.

4. Напишите название чемпиона, скина или ID скина.
5. Введите номер нужной строки. После успешной смены переоткройте страницу профиля в клиенте.

Если `league-bg.exe` скопирован в отдельную папку, запускайте его непосредственно из этой папки:

```bat
league-bg.exe
```

## Возможности

- Полный каталог скинов и чемпионов, а не только коллекция аккаунта.
- Актуальные чемпионы, включая Yunara.
- Поиск по русским и английским названиям, а также по ID скина.
- Поддержка Ahri Signature Immortalized Legend (Faker), ID `103087`.
- Проверка авторизации LCU перед отображением каталога.
- Автономный `.exe` для Windows: конечному пользователю не требуется Node.js.

## Ошибки

### `LCU authorization failed` или HTTP `401`

Закройте утилиту, убедитесь, что клиент League полностью загрузился и вы вошли в аккаунт, затем запустите `league-bg.exe` снова. Утилита читает активные данные подключения из lockfile клиента.

### Клиент League не найден

Запустите League of Legends и дождитесь главного окна клиента. Запускайте утилиту от того же пользователя Windows, от которого запущен League.

### Скин не найден

Проверьте написание или введите числовой ID скина. Каталог загружается из CommunityDragon, поэтому для запуска требуется подключение к интернету.

## Сборка из исходников

Этот раздел нужен только разработчикам. Для сборки требуется Node.js 20 или новее.

```bat
npm install
npm run build:win
```

Готовый файл будет создан по пути `dist/league-bg.exe`.

## Примечания

LCU — локальный интерфейс клиента, который Riot может изменять без предварительного уведомления. Утилита не изменяет игровые файлы и не вмешивается в игровой процесс.
